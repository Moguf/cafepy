# cafepy(Python3 scripts)
These scripts are for analyzing CafeMol outputs.

## Requirements
- setuptools >= 20.7

## Set Up
```bash
python3 -m setuptools numpy scipy matplotlib

```
## build & install
```
cd scipy
python3 setup.py build
python3 setup.py install
```

## Example:
```bash
cafepy com -f filename.dcd -o output.file

```
